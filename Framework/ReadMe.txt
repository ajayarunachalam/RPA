To run the Framework:-
	



1) Install flask, if not installed. To install flask run:- "sudo pip install flask" on the terminal.

	

2) Navigate to this directory, i.e., "Framework". 

After that, run: "sudo python3 framework.py". You will see that flask server is running.
	

3) 

Next,
 open browser and enter the url 		


http://127.0.0.1:5000/ 

4) Other bundled user-functionalities include:-


	a) http://127.0.0.1:5000/  /* Display main page with dashboard
	
*/
	b) http://127.0.0.1:5000/show /* Display all data without any GUI interface
 */	

	c) http://127.0.0.1:5000/drop/admin /* Here, admin is default username (that can be changed in framework.py), use this url to drop the table */
